Project 10 - Map location and coordinates

In your individual web site, enhance your existing page about browser security to show users their current location on a map. Note that you must specify a height and width using CSS for the element in which you display the map; these dimensions can be any size you choose. Also enhance the page to display the user's latitude, longitude, and altitude, with a label for each value.

Rather than display the current altitude at the device location, you can move the map and click on a point to display the altitude there - I did this partially because it was just easier, partially because position.coords.altitude returns null, but mostly because my map focused about a half mile south of my actual location and was giving me the wrong elevation.
