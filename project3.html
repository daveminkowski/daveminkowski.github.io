Project 3 - Functionality with simple flow control:

Plan and add a feature to one of the web pages in your personal site that incorporates content or functionality created by a series of if, if/else, and/or else if statements, or by a switch statement. If your page requires an event listener, create one that's backward compatible with older versions of Internet Explorer. View and test your page in one or more browsers as appropriate to ensure it works as you expect.

Simple switch selector:

Click one of these football team buttons. A link to the team's official homepage will appear just above an image of their logo.

There's a single placeholder image element below, hidden until you click a button. Basically, the switch statement uses the ID of the last-clicked button to determine which image should be used as the <img src> for the invisible placeholder image, then displays the appropriate image.
